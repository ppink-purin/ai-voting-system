# 컨셉 1: Futuristic Glassmorphism (미래지향적 글래스모피즘)

## 디자인 컨셉 개요
투명한 유리 효과와 그라디언트 배경을 활용한 미래지향적 디자인으로, AI 혁신이라는 테마와 완벽하게 조화를 이루는 컨셉입니다.

## 선정 사유

### 1. AI/기술 테마와의 완벽한 조화
- Glassmorphism은 2025년에도 여전히 인기 있는 트렌드로, 특히 AI 및 기술 스타트업 분야에서 선호됩니다
- 반투명한 유리 효과는 미래지향적이고 혁신적인 느낌을 제공합니다
- 그라디언트 배경은 AI의 다층적이고 복잡한 특성을 시각적으로 표현합니다

### 2. 시각적 깊이감과 현대성
- 블러 효과와 투명도를 통해 레이어 간의 깊이감을 자연스럽게 표현
- 세련되고 고급스러운 느낌으로 참여자들에게 긍정적인 인상 제공
- 현대적인 CSS 기술(backdrop-filter)을 활용한 최신 웹 트렌드 반영

### 3. 우수한 가독성과 사용성
- 반투명 카드 위의 텍스트는 충분한 대비를 유지하여 가독성 확보
- 그라디언트 배경이 각 카드의 경계를 명확히 구분
- 호버 효과를 통한 직관적인 인터랙션 제공

### 4. 행사의 프리미엄 이미지 구축
- 글래스모피즘은 고급스럽고 세련된 느낌을 전달
- AI 혁신 사례 발표라는 행사의 중요성과 가치를 시각적으로 강조
- 참여자들에게 특별한 경험을 제공

## 주요 디자인 요소

### 배경
- **그라디언트**: 보라색(#667eea) → 핑크(#f093fb) → 주황색(#f5576c)의 다채로운 그라디언트
- **애니메이션**: 부드러운 그라디언트 이동 효과로 동적인 느낌 제공

### 카드 스타일
- **투명도**: `background: rgba(255, 255, 255, 0.15)`
- **블러 효과**: `backdrop-filter: blur(20px)` - 배경을 흐리게 처리
- **테두리**: `border: 1px solid rgba(255, 255, 255, 0.2)` - 미묘한 유리 경계선
- **그림자**: 부드러운 섀도우로 떠있는 느낌 연출

### 인터랙션
- **호버 효과**: 스케일 변화(1.02배), 밝기 증가, 섀도우 강화
- **선택 상태**: 밝은 파란색 테두리와 배경으로 명확한 시각적 피드백
- **부드러운 전환**: 모든 상태 변화에 0.3초 transition 적용

### 타이포그래피
- 큰 헤더 폰트(3xl)로 중요 정보 강조
- 흰색 텍스트로 그라디언트 배경과 높은 대비 제공
- 명확한 시각적 계층 구조

## 기술적 구현
- CSS `backdrop-filter`를 활용한 유리 효과
- Tailwind CSS의 그라디언트 유틸리티
- 커스텀 애니메이션으로 배경 그라디언트 이동
- 반응형 디자인으로 모든 디바이스 지원
