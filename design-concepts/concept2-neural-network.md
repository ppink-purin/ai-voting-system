# 컨셉 2: Neural Network Theme (신경망 네트워크 테마)

## 디자인 컨셉 개요
AI의 핵심인 신경망(Neural Network)을 시각적으로 표현한 디자인으로, 연결된 노드와 네트워크 패턴을 배경에 적용하여 AI 혁신의 본질을 직관적으로 전달합니다.

## 선정 사유

### 1. AI 테마의 직관적 시각화
- 신경망은 AI의 핵심 개념으로, 행사 주제와 직접적으로 연결됩니다
- 연결된 노드 패턴은 AI의 학습과 연결성을 상징적으로 표현
- 참여자들에게 AI 혁신 행사임을 즉각적으로 인식시킵니다

### 2. 사이버펑크 미학의 현대적 적용
- 어두운 배경(dark mode)에 네온 색상(보라, 시안, 핑크)의 조합
- 기술적이고 미래지향적인 분위기 조성
- 젊고 혁신적인 이미지로 참여자들의 관심 유도

### 3. 동적인 시각적 경험
- 애니메이션으로 움직이는 네트워크 라인과 노드
- 마우스 인터랙션에 반응하는 연결선 효과 (선택사항)
- 정적이지 않은 살아있는 인터페이스로 참여 유도

### 4. 명확한 시각적 계층
- 어두운 배경 위의 밝은 카드로 콘텐츠 영역 명확히 구분
- 네온 색상의 강조 요소로 중요 정보 하이라이트
- 우수한 대비로 가독성 확보

## 주요 디자인 요소

### 배경
- **베이스 색상**: 매우 어두운 네이비/블랙 (`#0a0e27`)
- **네트워크 패턴**: SVG 또는 Canvas로 구현된 연결 노드
- **노드 색상**: 시안(`#00d4ff`), 보라(`#b624ff`), 핑크(`#ff006b`)
- **애니메이션**: 노드와 연결선이 부드럽게 펄스/이동

### 카드 스타일
- **배경**: 어두운 반투명 (`rgba(15, 23, 42, 0.85)`)
- **테두리**: 네온 그라디언트 테두리 효과
- **그림자**: 네온 글로우 효과로 떠있는 느낌
- **호버 시**: 네온 테두리가 밝아지고 글로우 강화

### 선택 상태
- **활성화**: 밝은 시안 테두리 + 강한 글로우
- **체크마크**: 네온 색상의 체크 아이콘
- **배경 강조**: 살짝 밝아진 배경으로 선택 상태 강조

### 타이포그래피
- **헤더**: 큰 크기, 화이트 색상, 약간의 텍스트 섀도우
- **본문**: 밝은 그레이로 가독성 유지
- **강조**: 시안/보라 색상으로 중요 정보 하이라이트

### 특수 효과
- **스캔라인**: 미묘한 수평선 효과로 디지털/스크린 느낌
- **노이즈 텍스처**: 배경에 미세한 그레인 효과
- **글로우 애니메이션**: 선택된 카드에 펄스 효과

## 기술적 구현
- Canvas API 또는 SVG를 활용한 네트워크 배경 애니메이션
- CSS `box-shadow`를 활용한 네온 글로우 효과
- Tailwind CSS dark mode 유틸리티
- requestAnimationFrame을 활용한 부드러운 애니메이션
- 성능 최적화를 위한 적절한 애니메이션 제한

## 사용자 경험 고려사항
- **접근성**: 충분한 색상 대비 유지 (WCAG AA 이상)
- **성능**: 애니메이션이 저사양 디바이스에서도 부드럽게 작동
- **모션 감도**: prefers-reduced-motion 미디어 쿼리 지원
- **가독성**: 어두운 배경에서도 명확한 텍스트 표시
