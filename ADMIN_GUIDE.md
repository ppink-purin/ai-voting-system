# 관리자 사용 가이드

AI 혁신 발표 투표 시스템의 관리자 기능 사용 방법을 안내합니다.

## 접속 정보

### 로컬 환경
- **URL**: http://localhost:3000/admin/login
- **비밀번호**: `admin2024` (`.env.local`에서 변경 가능)

### 프로덕션 환경
- **URL**: https://your-app.vercel.app/admin/login
- **비밀번호**: Vercel 환경변수 `ADMIN_PASSWORD`에 설정한 값

## 관리자 로그인

1. `/admin/login` 페이지 접속
2. 관리자 비밀번호 입력
3. "로그인" 버튼 클릭

> **참고**: 비밀번호는 브라우저 sessionStorage에 저장되어 탭을 닫으면 자동으로 삭제됩니다.

## 대시보드 기능

### 1. 헤더 영역
- **새로고침 버튼**: 수동으로 통계 업데이트
- **로그아웃 버튼**: 관리자 세션 종료

### 2. 통계 요약
네 가지 주요 정보를 한눈에 확인:

#### 총 참여자
- 현재까지 투표한 총 참여자 수
- 최소 1개 이상 선택한 사용자 포함

#### 필요 선택 갯수
- 참여자가 선택해야 하는 팀의 갯수 (예: 5개)
- 설정 관리 섹션에서 변경 가능

#### 투표 상태
- **진행 중** (녹색): 참여자들이 투표 가능
- **종료됨** (빨간색): 투표가 종료되어 수정 불가

#### 투표 제어
- **투표 종료** 버튼: 진행 중일 때 표시
- **투표 시작** 버튼: 종료되었을 때 표시

### 3. 설정 관리

관리자 페이지에서 시스템 설정을 동적으로 변경할 수 있습니다.

#### 발표 데이터 업로드
- JSON 파일 선택 후 업로드
- 형식: `{"presentations": [{"id": 1, "teamName": "팀명", "title": "제목"}]}`
- 즉시 반영 (서버 재시작 불필요)

#### 필요 선택 갯수 설정
- 참여자가 선택해야 하는 팀의 갯수 지정 (1개 이상)
- 숫자 입력 후 "변경" 버튼 클릭
- 변경 시 기존 투표 무효화 알림 표시

### 4. 디자인 테마 설정

참여자 페이지의 디자인을 선택할 수 있습니다.

#### 테마 선택
- 6개 컨셉 중 하나 선택:
  1. Futuristic Glassmorphism (미래지향적 글래스모피즘)
  2. Neural Network Theme (신경망 네트워크 테마)
  3. Neural Network Theme (신경망 네트워크 테마)
  3. Minimalist Professional (미니멀 프로페셔널)
  4. Vibrant Innovation (생동감 넘치는 혁신)
  5. 3D Modern Cards (3D 모던 카드)
  6. **Cosmic Tech (우주 기술 테마)** ⭐ 기본값

#### 랜덤 디자인 스킨
- 체크박스 활성화 시: 참여자가 페이지를 열 때마다 무작위 테마 표시
- 비활성화 시: 선택한 테마 고정
- "테마 설정 적용" 버튼으로 저장

#### 컨셉 미리보기
- "모든 컨셉 미리보기 →" 링크 클릭하여 전체 테마 확인 가능

### 5. 순위 및 통계 테이블

모든 발표의 투표 현황을 선택 횟수 순으로 정렬하여 표시합니다.

| 컬럼 | 설명 |
|------|------|
| **번호** | 발표 ID |
| **팀명** | 발표 팀 이름 |
| **발표 제목** | 발표 주제 |
| **선택 횟수** | 해당 발표를 선택한 참여자 수 |
| **선택율** | 전체 참여자 대비 선택 비율 |

**정렬 옵션**:
- 파일 순서: 원본 순서대로 표시
- 투표수 ↓: 선택 횟수 내림차순
- 투표수 ↑: 선택 횟수 오름차순

### 6. 개별 투표 내역 테이블

모든 참여자의 상세 투표 내역을 확인할 수 있습니다.

| 컬럼 | 설명 |
|------|------|
| **참여자 ID** | 익명 세션 ID (앞 12자리만 표시) |
| **선택 갯수** | 해당 참여자가 선택한 팀의 갯수 |
| **선택한 발표 팀 ID** | 선택한 팀의 ID 목록 |

**예시**: `#1, #5, #12, #15, #18` → 1번, 5번, 12번, 15번, 18번 팀 선택

## 투표 초기화

모든 투표 데이터를 삭제하고 시스템을 초기 상태로 되돌립니다.

### 사용 방법
1. 관리자 대시보드 하단 "투표 데이터 초기화" 버튼 클릭
2. 경고 메시지 확인:
   ```
   ⚠️ 경고: 모든 투표 데이터가 삭제됩니다!
   참여자: X명, 투표: Y건
   정말로 초기화하시겠습니까?
   ```
3. "확인" 클릭 시 즉시 삭제

### 초기화 내용
- ✅ 모든 참여자 세션 삭제
- ✅ 모든 투표 데이터 삭제
- ✅ 통계 0으로 초기화
- ❌ 발표 데이터는 유지
- ❌ 시스템 설정 유지 (선택 갯수, 테마 등)

### 주의사항
- **복구 불가능**: 한 번 삭제하면 되돌릴 수 없습니다
- **행사 전 테스트**: 테스트 투표 후 초기화하여 깨끗한 상태로 시작
- **실수 방지**: 행사 진행 중에는 사용하지 말 것

## 문제 해결

   - 프로덕션: Vercel 환경변수
2. 서버 재시작 (로컬인 경우)
3. 브라우저 캐시 삭제 후 재시도

3. 브라우저 콘솔 에러 메시지 확인
